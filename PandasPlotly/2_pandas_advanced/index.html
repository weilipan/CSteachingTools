⚗️ 單元三：欄位工程與邏輯運算 特徵工程 + 等高長條圖

📚 Pandas 欄位操作指南
① 增加欄位與純運算

將 Attack 與 Defense 相加，建立新欄位 Total_Stats。

# 新增 Total_Stats 欄位 df['Total_Stats'] = df['Attack'] + df['Defense']
Name Attack Defense
② 邏輯判斷與百分比運算

為了繪製等高長條圖，需將 Attack 與 Defense 轉換為「佔兩者總和的百分比」。

# 計算各佔 Total_Stats 的百分比 df['Atk_pct'] = (df['Attack'] / df['Total_Stats'] * 100).round(1) df['Def_pct'] = (df['Defense'] / df['Total_Stats'] * 100).round(1)
Name Attack Defense
③ 視覺化：等高長條圖（100% Stacked Bar Chart）

每列長條圖總長度相同（100%），用顏色對比攻擊與防禦的比例分布。

import plotly.express as px fig = px.bar(df, y='Name', x=['Atk_pct', 'Def_pct'], orientation='h', barmode='stack', title='攻防比例（等高長條圖）') fig.show()
