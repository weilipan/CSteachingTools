🍽️ 模組一：【上菜前的擺盤】欄位重命名

df.rename(columns=change) — 把難以辨識的英文欄位名稱，換成一目了然的中文。

# 定義對照字典
change = {
'electricity_generation': '總發電量',
'iso_code': '國家代碼',
'country': '國家',
'year': '年份'
}
df.rename(columns=change, inplace=True)

⚠️ Pandas 的禁忌： Pandas 是整批向量判斷，and / or 無法處理整個 Series。
請改用 &（AND）、|（OR）、~（NOT）進行位元邏輯運算！

🎨 模組五：【視覺化饗宴】Plotly 互動折線圖

color 參數不只是塗顏色，更是強大的自動分群 (Groupby) 魔法！

import plotly.express as px
 
# ① 未設定 color (混亂!)
fig = px.line(
df,
x='year',
y='life_expectancy',
# color='國家' ← 還沒加
)
fig.show()
💡 color 的真正魔力：
不只是塗顏色！Plotly 會自動依 color 欄位分組，為每個分類繪製獨立折線，並自動生成圖例 (Legend)。等同於先 groupby 再分別繪圖，一行搞定！
