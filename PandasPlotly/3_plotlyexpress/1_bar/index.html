長條圖與堆疊
px.bar

以「職業籃球隊伍場均得分 (2015–2025)」為主線，從基礎繪圖、資料切分到動態時間軸，完整掌握 px.bar 的每一個進階技法。

📊 px.bar 官方 API 文件 🐼 pandas plot.bar
📋 M1–M2：資料認識與基礎長條圖

準備一份包含年份、球隊與前/後場得分組成的資料表。使用最精簡的指令繪製總得分長條圖。

📄 DataFrame 預覽 (df) 節錄
#年份球隊位置得分球星

ℹ️ 每支球隊的總得分是由「後場得分」與「前場得分」加總而成，這也是後續展示「堆疊(stack)」的核心邏輯。
💻 基礎程式碼
import plotly.express as px df_2025 = df[df['年份'] == 2025] fig = px.bar(df_2025, x='球隊', y='得分') fig.show()

🎨 M3–M5：標籤、資料切分與堆疊模式

加入標題，並使用 color 將總得分拆解為前場/後場得分，最後切換 barmode 比較「堆疊」與「並排」的視覺差異。

💻 程式碼
fig = px.bar(df_2025, x='球隊', y='得分', title='2025 聯盟火力分析<br><sup>前/後場得分佔比</sup>', labels={'得分':'場均總分'}, color='位置', color_discrete_map={'後場':'#1f77b4', '前場':'#ff7f0e'}, barmode='stack' )

🖱️ M6–M7：懸停資訊與水平長條圖

加入 Tooltip 顯示詳細球星數據。當 X 軸標籤太擠時，使用 orientation='h' 將圖表轉為水平，並將 x 與 y 欄位對調。

💻 程式碼
fig = px.bar(df_2025, x='球隊', y='得分', color='位置', hover_name='球星', hover_data={'命中率':True}, orientation='h' # 記得 X 與 Y 需對調！ )

✏️ M8–M9：參考線與圖表註解

在圖表繪製完成後，對 fig 進行後處理，加入明確的資料比較基準，讓圖表更具洞察力。

💻 程式碼
fig.add_hline( y=116.4, line_dash='dash', line_color='red' ) fig.add_hrect( y0=120, y1=130, fillcolor='green', opacity=0.15 ) fig.add_annotation( x='塞爾提克', y=122.5, text='聯盟最高' )
