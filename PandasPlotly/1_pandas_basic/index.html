🔪 模組三：數字座標切蛋糕 iloc 精準定位

iloc[列位置, 欄位置] 認數字不認人，從 0 開始計算。切片 [A:B] 不包含 結尾的 B。

🧬 模組八：資料型態轉換 dtype & astype

📚 Series.dtype 官方文件

理解每一欄的資料型別 (dtype) 非常重要。字串數字無法運算，帶有 NaN 的整數欄位預設會被迫轉為浮點數 (float64)。特別注意：轉換型態後必須「重新賦值」才會生效！

🔍 模組九：多重條件篩選 符號與括號的鐵則

在 Pandas 中，多條件篩選必須使用位元運算子 & (且)、| (或)、~ (反)。絕對不能使用 Python 原生的 and、or、not，否則會觸發 ValueError。此外，每個獨立條件都必須用圓括號 () 包裝起來！

🚀 模組十二：終極實戰演練 從髒資料到乾淨洞察

📄 data_pipeline.py
import pandas as pd df = pd.read_csv( 'energy_data.csv', encoding='utf-8-sig', skiprows=3, header=0 ) df.rename(columns=str.lower, inplace=True) df = df.convert_dtypes() df.dropna(how='all', inplace=True) df.fillna(0, inplace=True) df.drop_duplicates(inplace=True) df = df[(df['electricity'] > 0) & (df['year'] >= 2020)] summary = df.describe() print(summary)
