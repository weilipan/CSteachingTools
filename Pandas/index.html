1. 建立 DataFrame (Creating Data)

首先，我們需要引入 pandas 庫。最常見的核心物件是 DataFrame，它就像是一個強大的 Excel 表格。

  • pd.DataFrame(data): 將字典或陣列轉換為表格。
  • 字典的 Key 會變成欄位名稱 (Column Name)。
[ ]:
import pandas as pd

# 準備數據
data = {
  'Name': ['Alice', 'Bob', 'Charlie', 'David'],
  'Age': [25, 30, 35, 28],
  'City': ['New York', 'Los Angeles', 'Chicago', 'Houston'],
  'Score': [85, 92, 78, 95]
}

# 建立 DataFrame
df = pd.DataFrame(data)
df # 顯示表格
小撇步： 在 Colab/Jupyter 中，直接在 Cell 最後一行打變數名稱（如 df），會自動印出漂亮的表格，不需要用 print(df)。

2. 快速檢視數據 (Inspection)

拿到數據後，我們先檢查此數據的基本樣貌。

常用方法：

  • df.head(n): 看前 n 筆資料 (預設 5)。
  • df.tail(n): 看後 n 筆資料。
  • df.info(): 查看資料型態、是否有缺失值。
  • df.describe(): 查看數值欄位的統計資訊 (平均、標準差、最大最小)。
[ ]:
# 查看前 2 筆數據
df.head(2)
[ ]:
# 查看資料結構與型態
df.info()
[ ]:
# 快速統計數據分佈
df.describe()

3. 選取與過濾 (Selection & Filtering)

精準提取我們需要的資料。

  • df['欄位名']: 選取單一欄位 (Series)。
  • df[['欄A', '欄B']]: 選取多個欄位。
  • df[條件]: 條件過濾，例如 df[df['Score'] > 90]。
  • .loc[]: 透過「標籤名稱」選取。
  • .iloc[]: 透過「索引位置(第幾列/第幾行)」選取。
[ ]:
# 1. 單一條件：找出 Age 大於 28 的人
df[df['Age'] > 28]
[ ]:
# 2. 多重條件：找出 Age > 25 且 Score >= 90 (使用 & 及 | )
# 注意：每個條件都要用括號 () 包起來
df[(df['Age'] > 25) & (df['Score'] >= 90)]
[ ]:
# 3. 選取特定欄位
df[['Name', 'Score']]
[ ]:
# 4. 使用 .iloc 切片 (選取第 1 到 3 列，第 0 到 2 行)
# 語法：df.iloc[列範圍, 行範圍] (不包含結束點)
df.iloc[1:3, 0:2]

4. 分組統計 (Grouping)

groupby 是數據分析的神器，類似 SQL 的 GROUP BY。

[ ]:
# 依城市分組，並計算分數平均值
df.groupby('City')['Score'].mean()
