使用 plotly.express 打造互動式圖表

Plotly Express 是 Python 中一個高層次的資料視覺化庫，它語法簡潔、功能強大，能輕鬆產出互動性極高的圖表。 本指南將帶你快速掌握最常用的五種圖表類型。

長條圖 Bar Chart

用於比較不同類別之間的數值大小。

Python 
import plotly.express as px

# 準備資料
data = {
    "City": ["台北", "台中", "高雄", "台南"],
    "Population": [250, 280, 270, 180]
}

# 繪製長條圖
fig = px.bar(
    data, 
    x="City", 
    y="Population", 
    title="台灣主要城市人口數",
    color="City"  # 根據城市上色
)
fig.show()

直方圖 Histogram

用於顯示連續數據的分布情況，了解數據的集中趨勢與離散程度。

Python 
import plotly.express as px
import numpy as np

# 產生隨機數據 (假設為學生成績)
scores = np.random.normal(70, 10, 200)

# 繪製直方圖
fig = px.histogram(
    x=scores, 
    nbins=20, 
    title="學生成績分布直方圖",
    labels={'x':'分數', 'y':'人數'},
    color_discrete_sequence=['#ff7f0e']
)
fig.show()

折線圖 Line Chart

用於顯示數據隨時間變化的趨勢，適合觀察波動與成長。

Python 
import plotly.express as px

# 準備時間序列資料
data = {
    "Month": ["Jan", "Feb", "Mar", "Apr", "May", "Jun"],
    "Sales": [100, 120, 150, 170, 160, 200],
    "Profit": [20, 30, 40, 50, 45, 70]
}

# 繪製折線圖
fig = px.line(
    data, 
    x="Month", 
    y=["Sales", "Profit"], 
    title="上半年營收與利潤趨勢",
    markers=True # 顯示數據點
)
fig.show()

堆疊圖 Stacked Bar Chart

用於顯示各類別內部的組成結構，同時比較總量。

Python 
import plotly.express as px

# 準備分類資料
data = {
    "Year": ["2023", "2023", "2023", "2024", "2024", "2024"],
    "Product": ["Phone", "Laptop", "Tablet", "Phone", "Laptop", "Tablet"],
    "Revenue": [500, 300, 150, 600, 350, 200]
}

# 繪製堆疊長條圖
fig = px.bar(
    data, 
    x="Year", 
    y="Revenue", 
    color="Product", # 根據產品堆疊
    title="年度產品營收結構",
    barmode="stack" # 預設即為 stack, 也可以用 'group'
)
fig.show()

圓餅圖 Pie Chart

用於顯示各部分佔總體的比例。

Python 
import plotly.express as px

# 準備資料
data = {
    "Browser": ["Chrome", "Edge", "Firefox", "Safari", "Others"],
    "Usage": [65, 15, 10, 8, 2]
}

# 繪製圓餅圖
fig = px.pie(
    data, 
    values="Usage", 
    names="Browser", 
    title="瀏覽器市佔率",
    hole=0.4 # 設定中空大小 (變成甜甜圈圖)
)
fig.show()
