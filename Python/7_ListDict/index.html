🚫 模組 4：誰能當鑰匙？ — 不可變法則 (Hashable)
Dict 的 Key 必須是「不可變 (Immutable)」的。字串、整數可以；但容易變形的 List 絕對不行！

🪆 模組 5：俄羅斯套娃實戰 — JSON 結構解析
最常見的 List of Dicts 結構。先用「數字索引」開大鐵櫃，再用「字串 Key」開保險箱。
users = [ {"id": 1, "name": "Alice"}, {"id": 2, "name": "Bob"} ] print( users[1]["name"] )

🏆 模組 6：綜合實戰 — 社群媒體貼文過濾系統
融合 for 迴圈、if 判斷與兩種資料結構。點擊「下一步」追蹤嚴格 ind-3 縮排防護下的程式碼！
popular_posts = [] for post in posts: if post["likes"] > 100: popular_posts.append(post["author"]) print(popular_posts)
