5獨立 vs 互斥 — if/if vs if/elif 的陷阱

消費金額：$250。觀察兩種寫法的不同結果。

❌ 兩個獨立 if（錯）

if amount >= 100: send_coupon(1) if amount >= 200: send_coupon(2)

✅ if + elif（正確）

if amount >= 200: send_coupon(2) elif amount >= 100: send_coupon(1)
