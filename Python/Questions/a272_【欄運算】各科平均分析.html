📊 二維串列 (2D List) 運作模擬器

📝 題目：各科平均分析

內容：

這是資料分析最常見的需求：計算「每一個欄位」的平均值。輸入一份 N 位學生的成績表，固定有 3 個科目：國文、英文、數學。請計算並輸出這三科的班級平均分數。

輸入說明：

第一行：一個整數 N (學生數)。
接下來 N 行：每行 3 個整數 (順序為：國、英、數)。

輸出說明：

輸出一行，包含 3 個浮點數，代表國、英、數的平均，請保留一位小數，中間以空白隔開。

💡 語法小提示：Pythonic 的輸入技巧
程式碼第 5 行的 list(map(int, input().split())) 是解題系統中最常見的寫法。
它的作用完全等同於串列推導式 (List Comprehension)：[int(i) for i in input().split()]。
兩者都能將讀取到的一整行字串，瞬間切割並轉換成整數串列 (例如：[100, 100, 100]) 喔！
