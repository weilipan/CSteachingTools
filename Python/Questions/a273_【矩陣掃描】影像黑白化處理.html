📷 影像二值化：巢狀矩陣掃描模擬

📝 題目敘述：影像黑白化處理

內容：
電腦中的灰階圖片其實就是一個二維陣列 (Matrix)，每個格子 (Pixel) 代表亮度 (0~255)。
在影像辨識中，我們常做「二值化 (Binarization)」：設定一個門檻值 T，亮度超過 T 的變 1 (白)，否則變 0 (黑)。
請讀入一個 R x C 的矩陣，並輸出二值化後的結果。

輸入說明：
第一行：三個整數 R (列), C (欄), T (門檻值)。
接下來 R 行：每行 C 個整數 (代表亮度)。

輸出說明：
輸出 R 行，每行 C 個整數 (只包含 0 或 1)，以空白隔開。

💡 語法提示：Pythonic 輸入技巧

  • 處理二維陣列的輸入時，list(map(int, input().split())) 是非常強大的標準寫法。
  • 它的作用完全等同於串列推導式 (List Comprehension)：
    [int(i) for i in input().split()]。
  • 透過這個技巧，我們可以一次將一整排由空白隔開的灰階數值，直接轉換成一個整數陣列塞進 image 中！
