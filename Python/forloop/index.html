核心概念

🔍 可迭代物件 (Iterable)

for 迴圈可以遍歷任何「可迭代」的物件，例如列表 (List)、字串 (String)、元組 (Tuple) 或字典 (Dictionary)。 每次迴圈執行時，會自動取出下一個元素。

🔢 range() 函式

當你需要執行特定次數，或生成數字序列時，常用 range(start, stop, step)。 注意：stop 是不包含的（例如 range(5) 產生 0 到 4）。

🛑 break 與 continue

break：直接強制結束整個迴圈。
continue：跳過「這一次」迭代的剩餘程式碼，直接進入下一次迭代。
