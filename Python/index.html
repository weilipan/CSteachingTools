Python 基礎開發指南

涵蓋從環境輸入輸出到核心控制流程的技術文件與視覺化模擬。

01
Basic Input / Output (基本輸入輸出) 學習 print() 與 input()，掌握標準輸入輸出與 f-string 格式化。
02
Control Flow: If-Else (條件判斷) 學習 if, elif, else 條件判斷式、邏輯運算子與程式的分岔處理。
03
Iteration: For Loop (迴圈控制) 利用 for 迴圈與 range() 函數，讓電腦自動幫你執行重複的任務。
04
String Manipulation (字串處理) 深入字串切片 (Slicing)、常用字串方法與進階文字處理技巧。
