01. If 判斷式

單向選擇

最基本的判斷結構。如果條件成立 (True)，則執行縮排區塊內的程式碼；否則跳過。

score = 60
if score >= 60:
    print("及格！")

02. If-Else 判斷式

雙向選擇

二選一的結構。條件成立時執行 If 區塊，不成立時執行 Else 區塊。

password = "1234"
if password == "1234":
    print("登入成功")
else:
    print("密碼錯誤")

03. If-Elif-Else 多重判斷

多向選擇

當有多個條件需要檢查時使用。程式會由上而下檢查，一旦某個條件成立，執行對應區塊後即離開整個結構。

age = 15
if age < 12:
    print("兒童票")
elif age < 65:
    print("全票")
else:
    print("敬老票")
Age < 12?
True → 兒童票
False ↓
Age < 65?
True → 全票
False ↓
Else: 敬老票

04. 巢狀判斷 (Nested If)

進階結構

在一個判斷式內部再包含另一個判斷式。就像剝洋蔥一樣，層層檢查。

has_ticket = True
is_vip = False

if has_ticket:
    if is_vip:
        print("VIP 通道")
    else:
        print("一般通道")
else:
    print("請先購票")
Outer If (Has Ticket?)
Inner If (Is VIP?)
