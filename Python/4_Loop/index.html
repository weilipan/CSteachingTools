05 時鐘齒輪：巢狀迴圈 (Nested Loops)
for i in range(3): for j in range(4): print(i, j)
大齒輪 = 外層 i（轉3次）
小齒輪 = 內層 j（每次轉4步）
外層 i = -  |  內層 j = -
