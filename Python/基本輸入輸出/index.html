📤 print() 函數

print() 是 Python 中最基本的輸出函數，用於在控制台顯示資訊。

基本用法

print("Hello, World!") print("你好，世界！") print(123) print(3.14)

多個參數輸出

name = "小明" age = 20 print("姓名:", name, "年齡:", age)

sep 和 end 參數

# sep 控制分隔符號 print("A", "B", "C", sep="-") # end 控制結尾字元 print("第一行", end=" | ") print("第二行")

📥 input() 函數

input() 函數用於從使用者獲取輸入，返回的值永遠是字串型態。

基本輸入

name = input("請輸入你的名字: ") print("你好,", name)

數字輸入與轉換

# input() 返回字串，需要轉換成數字 age = int(input("請輸入年齡: ")) height = float(input("請輸入身高(cm): ")) print(f"你 {age} 歲，身高 {height} 公分")

計算機範例

num1 = float(input("第一個數字: ")) num2 = float(input("第二個數字: ")) result = num1 + num2 print(f"{num1} + {num2} = {result}")

✨ 字串格式化

Python 提供多種字串格式化方法，讓輸出更加靈活美觀。

f-string (推薦)

name = "小華" age = 25 score = 95.5 print(f"姓名: {name}") print(f"年齡: {age} 歲") print(f"成績: {score} 分")

f-string 運算與格式化

price = 1234.5678 # 運算 print(f"總價: {price * 1.05}") # 小數位數控制 print(f"價格: {price:.2f} 元") # 對齊與填充 print(f"編號: {42:05d}")

format() 方法

name = "小明" score = 87.5 print("學生: {}, 分數: {}".format(name, score)) print("分數: {1}, 學生: {0}".format(name, score)) print("成績: {s:.1f}".format(s=score))

% 格式化 (舊式)

name = "小美" age = 22 gpa = 3.87 print("姓名: %s" % name) print("年齡: %d 歲" % age) print("GPA: %.2f" % gpa)

🔢 資料型態

Python 的基本資料型態包括整數、浮點數、布林值和字串。

int (整數)

age = 25 count = -10 big_num = 1000000 print(type(age)) # <class 'int'> print(age + 5) # 30

✓ 整數沒有小數點

✓ 可以是正數或負數

✓ 支援基本運算

float (浮點數)

pi = 3.14159 temp = -5.5 price = 99.99 print(type(pi)) # <class 'float'> print(f"{pi:.2f}") # 3.14

✓ 帶有小數點的數字

✓ 適合精確計算

✓ 可以格式化顯示

bool (布林值)

is_student = True has_license = False print(type(is_student)) # <class 'bool'> print(5 > 3) # True print(10 == 20) # False

✓ 只有 True 或 False

✓ 用於條件判斷

✓ 比較運算結果

str (字串)

name = "Python" message = '你好' multi = """多行 字串""" print(type(name)) # <class 'str'> print(name[0]) # P print(len(name)) # 6

✓ 文字資料

✓ 用引號包圍

✓ 可以索引和切片

型態轉換

# 字串轉數字 num_str = "123" num = int(num_str) print(num + 10) # 133 # 數字轉字串 age = 25 text = str(age) print("我" + text + "歲") # 我25歲 # 浮點數轉整數 pi = 3.14 print(int(pi)) # 3
