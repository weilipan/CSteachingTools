🌟 3. else-if 階梯 vs 全部用 if（致命差異）

輸入分數後點擊「▶ 模擬執行」，觀察兩種寫法的執行路徑有何不同。

✅ 正確：else-if 寫法
if (score >= 90) {   cout << "A"; } else if (score >= 80) {   cout << "B"; } else if (score >= 70) {   cout << "C"; } else {   cout << "F"; } ⚡ 條件已滿足，直接跳出！
— 等待執行 —
❌ 錯誤：全部獨立 if 寫法
if (score >= 90) {   cout << "A"; } if (score >= 80) {   cout << "B"; } if (score >= 70) {   cout << "C"; } if (score < 70) {   cout << "F"; }
