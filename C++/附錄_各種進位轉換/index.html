進位系統概念

🔢 二進位 (Binary)

基數：2

只使用 0 和 1 兩個數字，是電腦內部儲存資料的基本方式。

範例：1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 11₁₀

🎯 八進位 (Octal)

基數：8

使用 0-7 八個數字，常用於Unix系統的檔案權限表示。

範例：17₈ = 1×8¹ + 7×8⁰ = 15₁₀

🌟 十進位 (Decimal)

基數：10

使用 0-9 十個數字，是人類日常使用的計數系統。

範例：123₁₀ = 1×10² + 2×10¹ + 3×10⁰

🚀 十六進位 (Hexadecimal)

基數：16

使用 0-9 和 A-F，常用於記憶體位址和顏色代碼。

範例：1F₁₆ = 1×16¹ + 15×16⁰ = 31₁₀

C++ 中的進位表示法

進位系統 前綴 範例 說明
二進位 0b 或 0B 0b1010 C++14 起支援
八進位 0 012 以 0 開頭
十進位 無 10 預設表示法
十六進位 0x 或 0X 0xA 字母不區分大小寫
💡 注意事項：
  • 二進位字面值需要 C++14 或更新版本
  • 八進位以 0 開頭，容易造成混淆，使用時需特別注意
  • 十六進位的字母 A-F 可以大寫或小寫

完整程式碼範例

number_conversion.cpp 
#include <iostream>
#include <bitset>
#include <iomanip>
using namespace std;

int main() {
    // 宣告同一個數值的不同進位表示
    int binary = 0b1010;      // 二進位
    int octal = 012;          // 八進位
    int decimal = 10;         // 十進位
    int hexadecimal = 0xA;    // 十六進位
    
    cout << "=== 進位轉換示範 ===" << endl;
    cout << "所有數值都等於十進位的 10" << endl << endl;
    
    // 輸出為十進位
    cout << "十進位輸出：" << endl;
    cout << "binary = " << binary << endl;
    cout << "octal = " << octal << endl;
    cout << "decimal = " << decimal << endl;
    cout << "hexadecimal = " << hexadecimal << endl << endl;
    
    // 以不同進位格式輸出
    int num = 255;
    cout << "數值 255 的各種進位表示：" << endl;
    cout << "二進位：" << bitset<8>(num) << endl;
    cout << "八進位：" << oct << num << endl;
    cout << "十進位：" << dec << num << endl;
    cout << "十六進位：" << hex << uppercase << num << endl;
    
    return 0;
}

常用轉換函數

std::stoi()

將字串轉換為整數，可指定進位基數

string binary = "1010";
int decimal = stoi(binary, nullptr, 2);  // 結果: 10

string hex = "FF";
int decimal2 = stoi(hex, nullptr, 16);   // 結果: 255

std::bitset

用於二進位操作和顯示

bitset<8> bits(42);
cout << bits << endl;           // 輸出: 00101010
cout << bits.to_ulong() << endl; // 輸出: 42

std::oct, std::dec, std::hex

設定輸出流的進位格式

int num = 100;
cout << oct << num << endl;  // 輸出: 144 (八進位)
cout << dec << num << endl;  // 輸出: 100 (十進位)
cout << hex << num << endl;  // 輸出: 64 (十六進位)

std::setbase()

動態設定輸出基數（需 #include <iomanip>）

int num = 42;
cout << setbase(8) << num << endl;   // 八進位
cout << setbase(10) << num << endl;  // 十進位
cout << setbase(16) << num << endl;  // 十六進位
