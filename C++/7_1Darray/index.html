陣列在記憶體中是連續排列的，每個元素佔 4 bytes（int）。索引值從 0 開始到 N-1。 修改下方任一格的值，體驗 arr[i] = value。

int arr[5] = {12, 45, 7, 88, 33}; // 索引從 0 開始 → arr[0] = 12, arr[4] = 33
