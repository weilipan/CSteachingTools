探索 std::fixed 與 std::setprecision 的奧秘

⚙️ 設定 (Settings)

main.cpp 

#include <iostream>
#include <iomanip>

int main() {
    double num = 123.456789;
    std::cout << std::setprecision(6) << num;
    return 0;
}
🖥️ Console Output
123.457

📚 詳細說明 (Explanation)

預設模式 (Default)

當沒有設定 fixed 或 scientific 時，setprecision(n) 設定的是最大有效位數 (Significant Digits)。

  • 如果整數部分位數大於 n，可能會切換成科學記號 (視編譯器實作而定)。
  • 會自動移除末尾無意義的 0。

std::fixed 模式

使用 std::fixed 後，setprecision(n) 設定的是小數點後的位數。

  • 無論數值大小，都會以定點數形式顯示。
  • 若小數位數不足，會補 0。

std::scientific 模式

使用 std::scientific 後，setprecision(n) 設定的是科學記號中小數點後的位數。

  • 數值總是顯示為科學記號形式 (e.g., 1.23e+02)。
