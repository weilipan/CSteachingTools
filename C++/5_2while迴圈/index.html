什麼是 While 迴圈？

while 迴圈是 C++ 中最基本的循環結構之一。它會根據一個布林條件（Boolean condition）來決定是否重複執行一段程式碼。只要條件為 true，迴圈就會一直執行下去。

就像是：只要天還在下雨，我就一直待在室內。

基本語法

while (布林運算式) {
    // 只要條件為 true，這裡的程式碼就會執行
    // ...
}
  • 入口檢查： 在執行迴圈本體前先檢查條件。
  • 次數不固定： 只要條件成立就繼續，不一定要預知執行次數。

互動式模擬器

int i = 0;
while (i < 5) {
    cout << i << " ";
    i++;
}

常見錯誤

1. 無窮迴圈 (Infinite Loop)

忘記更新迴圈變數，導致條件永遠為 true。

while (i < 5) {
    cout << i;
    // 漏掉了 i++!
}

2. 分號誤用

在 while 括號後多加了分號，這會讓迴圈執行一個空語句。

while (i < 5); // 這裡的分號是錯誤的
{
    i++;
}
