🚀 C++ 快速排序法 (Quick Sort) 視覺化

深入理解分治法的經典排序算法

📚 演算法介紹

快速排序法 (Quick Sort) 是一種高效的排序演算法，由英國電腦科學家 Tony Hoare 於 1960 年提出。它採用分治法 (Divide and Conquer) 的策略，透過選擇一個基準值 (pivot) 將資料分割成兩部分，然後遞迴地對這兩部分進行排序。

核心概念：

  • 選擇基準 (Pivot Selection)：從陣列中選擇一個元素作為基準值
  • 分割 (Partition)：將陣列重新排列，使得比基準小的元素在左邊，比基準大的在右邊
  • 遞迴排序 (Recursive Sort)：對左右兩個子陣列遞迴地執行快速排序

💻 C++ 程式碼實作


#include <iostream>
#include <vector>
using namespace std;

// 分割函數：將陣列分為兩部分
int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[high];    // 選擇最右邊的元素作為基準
    int i = low - 1;          // i 指向小於 pivot 的區域的最後一個元素
    
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr[i], arr[j]);  // 將小於 pivot 的元素移到左邊
        }
    }
    swap(arr[i + 1], arr[high]);   // 將 pivot 放到正確位置
    return i + 1;                   // 回傳 pivot 的位置
}

// 快速排序主函數
void quickSort(vector<int>& arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);  // 取得分割點
        
        quickSort(arr, low, pi - 1);         // 遞迴排序左半部
        quickSort(arr, pi + 1, high);        // 遞迴排序右半部
    }
}

// 主程式
int main() {
    vector<int> arr = {64, 34, 25, 12, 22, 11, 90};
    int n = arr.size();
    
    cout << "排序前: ";
    for (int num : arr) cout << num << " ";
    
    quickSort(arr, 0, n - 1);
    
    cout << "\n排序後: ";
    for (int num : arr) cout << num << " ";
    
    return 0;
}

📊 複雜度分析

⏱️ 時間複雜度 (Time Complexity)

最佳情況 (Best Case)：

$$T(n) = O(n \log n)$$

當每次分割都能將陣列平均分成兩半時，達到最佳效能。

推導過程：

設 $T(n)$ 為排序 $n$ 個元素所需時間：

$$T(n) = 2T\left(\frac{n}{2}\right) + O(n)$$

其中：

  • $2T\left(\frac{n}{2}\right)$：兩個子問題各有 $\frac{n}{2}$ 個元素
  • $O(n)$：分割操作需要掃描整個陣列

根據主定理 (Master Theorem)：

$$T(n) = O(n \log n)$$

平均情況 (Average Case)：

$$T(n) = O(n \log n)$$

在隨機資料下，平均每次分割接近平衡，期望時間複雜度為 $O(n \log n)$。

最壞情況 (Worst Case)：

$$T(n) = O(n^2)$$

當陣列已經排序或逆序，且每次都選擇最大或最小元素作為 pivot 時。

推導過程：

每次分割只能減少一個元素：

$$T(n) = T(n-1) + O(n)$$

展開遞迴式：

$$T(n) = n + (n-1) + (n-2) + \cdots + 1$$

$$T(n) = \frac{n(n+1)}{2} = O(n^2)$$

💾 空間複雜度 (Space Complexity)

最佳/平均情況：

$$S(n) = O(\log n)$$

遞迴呼叫堆疊的深度在平衡分割時為 $\log n$。

說明：

每次遞迴呼叫需要儲存：

  • 函數參數 (low, high)
  • 區域變數 (pivot, i, j)
  • 返回地址

遞迴深度 = $\log_2 n$

總空間 = $O(\log n)$

最壞情況：

$$S(n) = O(n)$$

當分割極度不平衡時，遞迴深度可達 $n$。

說明：

最壞情況下，每次只減少一個元素，遞迴深度為 $n$，因此空間複雜度為 $O(n)$。

原地排序 (In-place)：

快速排序是原地排序演算法，除了遞迴堆疊外，不需要額外的陣列空間。

📝 詳細步驟說明

1

選擇基準值 (Choose Pivot)

從陣列中選擇一個元素作為基準值。常見策略：

  • 最右元素：選擇 arr[high]（本實作採用）
  • 最左元素：選擇 arr[low]
  • 中間元素：選擇 arr[(low+high)/2]
  • 隨機選擇：隨機選擇一個元素
  • 三數取中：取首、中、尾三個元素的中位數
2

分割操作 (Partition)

將陣列重新排列，使得：

  • 所有小於 pivot 的元素移到 pivot 左邊
  • 所有大於 pivot 的元素移到 pivot 右邊
  • pivot 放在正確的最終位置

分割過程：

  1. 設定指標 i = low - 1（指向小於 pivot 區域的最後一個元素）
  2. 用指標 j 從 low 掃描到 high - 1
  3. 若 arr[j] < pivot，則 i++，並交換 arr[i] 和 arr[j]
  4. 最後交換 arr[i+1] 和 arr[high]，將 pivot 放到正確位置
3

遞迴排序 (Recursive Sort)

對分割後的兩個子陣列遞迴執行快速排序：

  • 左子陣列：quickSort(arr, low, pi - 1)
  • 右子陣列：quickSort(arr, pi + 1, high)

終止條件：

當 low >= high 時，表示子陣列只有 0 或 1 個元素，已經排序完成。

4

合併結果 (Combine)

由於是原地排序，不需要額外的合併步驟。當所有遞迴呼叫完成後，整個陣列已經排序完成。

關鍵特性：

  • 每個 pivot 都已在其最終位置
  • 左子陣列的所有元素 < pivot < 右子陣列的所有元素
  • 遞迴完成後，整體有序

⚖️ 優缺點分析

✅ 優點

  • 平均效能優秀：平均時間複雜度為 $O(n \log n)$，實際執行速度快
  • 原地排序：不需要額外的陣列空間（除了遞迴堆疊）
  • 快取友善：具有良好的局部性，對現代 CPU 快取友善
  • 實用性高：在實務中表現優於其他 $O(n \log n)$ 演算法
  • 易於平行化：左右子陣列可以獨立處理

❌ 缺點

  • 不穩定排序：相同元素的相對順序可能改變
  • 最壞情況差：最壞情況下時間複雜度為 $O(n^2)$
  • 遞迴開銷：需要額外的堆疊空間
  • 對小陣列效率低：對於小規模資料，插入排序可能更快
  • Pivot 選擇影響大：不良的 pivot 選擇會嚴重影響效能

🔧 優化技巧

1. 三數取中法 (Median-of-Three)

選擇首、中、尾三個元素的中位數作為 pivot，避免最壞情況。

int medianOfThree(vector<int>& arr, int low, int high) {
    int mid = low + (high - low) / 2;
    if (arr[low] > arr[mid]) swap(arr[low], arr[mid]);
    if (arr[low] > arr[high]) swap(arr[low], arr[high]);
    if (arr[mid] > arr[high]) swap(arr[mid], arr[high]);
    return mid;
}

2. 小陣列使用插入排序

當子陣列大小小於某個閾值（如 10）時，改用插入排序。

void quickSort(vector<int>& arr, int low, int high) {
    if (high - low < 10) {
        insertionSort(arr, low, high);
        return;
    }
    // ... 正常的快速排序邏輯
}

3. 尾遞迴優化

先處理較小的子陣列，減少堆疊深度。

void quickSort(vector<int>& arr, int low, int high) {
    while (low < high) {
        int pi = partition(arr, low, high);
        if (pi - low < high - pi) {
            quickSort(arr, low, pi - 1);
            low = pi + 1;
        } else {
            quickSort(arr, pi + 1, high);
            high = pi - 1;
        }
    }
}

4. 三路分割 (3-Way Partition)

將陣列分成小於、等於、大於 pivot 三部分，處理重複元素更有效率。

// 適用於有大量重複元素的情況
void quickSort3Way(vector<int>& arr, int low, int high) {
    if (low >= high) return;
    int lt = low, gt = high, i = low + 1;
    int pivot = arr[low];
    while (i <= gt) {
        if (arr[i] < pivot) swap(arr[lt++], arr[i++]);
        else if (arr[i] > pivot) swap(arr[i], arr[gt--]);
        else i++;
    }
    quickSort3Way(arr, low, lt - 1);
    quickSort3Way(arr, gt + 1, high);
}
