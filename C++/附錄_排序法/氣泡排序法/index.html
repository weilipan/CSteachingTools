演算法步驟

void bubbleSort(int arr[], int n) {
  for (int i = 0; i < n-1; i++) {     
    for (int j = 0; j < n-i-1; j++) { 
      if (arr[j] > arr[j+1]) {        
        swap(&arr[j], &arr[j+1]);     
      }
    }
  }
}

複雜度分析

時間複雜度 (Time Complexity)

  • 最差情況 (Worst Case): $O(n^2)$
    當陣列為反向排序時。
  • 平均情況 (Average Case): $O(n^2)$
    需進行約 $n(n-1)/2$ 次比較。
  • 最佳情況 (Best Case): $O(n)$
    當陣列已經排序時 (需優化版)。
計算方式：

外層迴圈執行 $n-1$ 次，內層迴圈分別執行 $n-1, n-2, ..., 1$ 次。

總比較次數 = $(n-1) + (n-2) + ... + 1 = \frac{n(n-1)}{2} \approx O(n^2)$

空間複雜度 (Space Complexity)

  • $O(1)$
    只需要一個額外的暫存變數進行交換，屬於原地排序 (In-place Sort)。
