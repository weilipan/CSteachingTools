選擇排序法視覺化 Selection Sort

未排序 目前最小值 掃描中 已排序
準備就緒

C++ 程式碼

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int min_idx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx])
                min_idx = j;
        }
        swap(arr[min_idx], arr[i]);
    }
}

複雜度分析

時間複雜度 (Time Complexity) \( O(n^2) \)

選擇排序法需要兩層迴圈：

  • 外層迴圈執行 \( n-1 \) 次。
  • 內層迴圈分別執行 \( n-1, n-2, \dots, 1 \) 次。

總比較次數為等差級數總和：

\[ \sum_{i=1}^{n-1} i = \frac{n(n-1)}{2} \approx \frac{n^2}{2} \]

因此時間複雜度為 \( O(n^2) \)。無論陣列是否已排序，比較次數皆固定。

空間複雜度 (Space Complexity) \( O(1) \)

僅需少數變數（如 min_idx, i, j, temp）來儲存索引與暫存值。

排序過程在原陣列中進行（In-place），不需要額外的記憶體空間來儲存陣列副本。
