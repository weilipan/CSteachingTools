C++ Range-Based For Loop

探索現代 C++ 最優雅的迭代方式

什麼是 Range-Based For Loop？

Range-based for loop 是 C++11 引入的現代化語法，讓我們能夠更簡潔、更安全地遍歷容器中的元素。 它消除了傳統 for 迴圈中容易出錯的索引管理，讓程式碼更加清晰易讀。

基本語法

for (declaration : range_expression) {
    // 迴圈主體
}

語法比較

傳統 For Loop

int arr[] = {1, 2, 3, 4, 5};
for (int i = 0; i < 5; i++) {
    cout << arr[i] << " ";
}
❌ 需要管理索引

Range-Based For Loop

int arr[] = {1, 2, 3, 4, 5};
for (int num : arr) {
    cout << num << " ";
}
✅ 簡潔且安全

參考型別 (Reference)

使用參考型別可以直接修改容器中的元素，而不是複製它們。這在處理大型物件或需要修改元素時特別有用。

複製 (Copy)

int arr[] = {1, 2, 3};
for (int num : arr) {
    num = num * 2;  // 只修改副本
}
// arr 仍然是 {1, 2, 3}
❌ 原始陣列不會改變

參考 (Reference)

int arr[] = {1, 2, 3};
for (int& num : arr) {
    num = num * 2;  // 修改原始元素
}
// arr 變成 {2, 4, 6}
✅ 原始陣列被修改

互動式範例

常數參考 (Const Reference)

使用 const 參考可以避免複製開銷，同時防止意外修改元素。這是遍歷唯讀資料的最佳實踐。

效能比較

語法 複製開銷 可修改 適用場景
for (Type elem : container) 高 ❌ 小型基本型別
for (Type& elem : container) 無 ✅ 需要修改元素
for (const Type& elem : container) 無 ❌ 唯讀大型物件

實際範例

vector<string> names = {"Alice", "Bob", "Charlie"};

// 推薦：使用 const 參考
for (const string& name : names) {
    cout << name << endl;
    // name = "Test";  // 編譯錯誤！
}

Auto 關鍵字

使用 auto 關鍵字可以讓編譯器自動推導型別，讓程式碼更簡潔且易於維護。

基本用法

vector<int> numbers = {1, 2, 3};
for (auto num : numbers) {
    cout << num;
}

Auto 參考

vector<int> numbers = {1, 2, 3};
for (auto& num : numbers) {
    num *= 2;
}

Auto Const 參考

vector<string> names = {"A", "B"};
for (const auto& name : names) {
    cout << name;
}
✨ 最佳實踐

支援的容器類型

Range-based for loop 可以用於任何支援迭代器的容器，包括標準庫容器和 C 風格陣列。

C 風格陣列

int arr[] = {1, 2, 3};
for (auto num : arr) {
    cout << num;
}

Vector

vector<int> vec = {1, 2, 3};
for (auto num : vec) {
    cout << num;
}

String

string str = "Hello";
for (auto ch : str) {
    cout << ch;
}

Map

map<string, int> m;
for (const auto& pair : m) {
    cout << pair.first;
}

Set

set<int> s = {1, 2, 3};
for (auto num : s) {
    cout << num;
}

List

list<int> lst = {1, 2, 3};
for (auto num : lst) {
    cout << num;
}

常見陷阱與最佳實踐

⚠️

陷阱：修改容器大小

vector<int> vec = {1, 2, 3};
for (auto num : vec) {
    vec.push_back(num);  // 危險！
}

在迭代過程中修改容器大小會導致未定義行為

⚠️

陷阱：忘記使用參考

vector<LargeObject> objects;
for (auto obj : objects) {  // 複製！
    // 處理 obj
}

大型物件會被複製，造成效能問題

✅

最佳實踐：使用 const auto&

vector<string> names;
for (const auto& name : names) {
    cout << name;
}

預設使用 const 參考，除非需要修改

✅

最佳實踐：結構化綁定 (C++17)

map<string, int> scores;
for (const auto& [name, score] : scores) {
    cout << name << ": " << score;
}

使用結構化綁定讓程式碼更清晰

實戰範例

範例 1：計算陣列總和

#include <iostream>
using namespace std;

int main() {
    int numbers[] = {10, 20, 30, 40, 50};
    int sum = 0;
    
    for (const auto& num : numbers) {
        sum += num;
    }
    
    cout << "總和：" << sum << endl;  // 輸出：150
    return 0;
}

範例 2：字串處理

#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello";
    
    for (auto& ch : text) {
        ch = toupper(ch);
    }
    
    cout << text << endl;  // 輸出：HELLO
    return 0;
}

範例 3：Vector 操作

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> nums = {1, 2, 3, 4, 5};
    
    for (auto& num : nums) {
        num *= num;  // 平方
    }
    
    for (const auto& num : nums) {
        cout << num << " ";  // 1 4 9 16 25
    }
    return 0;
}

快速參考指南

for (Type elem : container)
複製元素（適用於小型型別）
for (Type& elem : container)
參考元素（可修改）
for (const Type& elem : container)
常數參考（唯讀，推薦）
for (auto elem : container)
自動型別推導（複製）
for (auto& elem : container)
自動型別推導（參考）
for (const auto& elem : container)
自動型別推導（常數參考，最推薦）
