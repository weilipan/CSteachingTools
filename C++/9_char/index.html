📦 <cctype> 函式庫

toupper(char c)
將小寫字母轉為大寫；非字母直接返回原值
tolower(char c)
將大寫字母轉為小寫；非字母直接返回原值
使用方式：
#include <cctype>
char r = toupper('a'); // → 'A'
🔍 字元判斷函式 <cctype> Library

選擇一個篩選函式，程式會逐一掃描下方字元，符合者亮綠色、不符合者亮紅色。

⚠️ 初學者大坑：單引號 vs 雙引號 'A' vs "A"

char c = 'A';

char c = 'A';

char s[] = "A";

char s[] = "A";
🚨
編譯錯誤警告：
char c = "A";  ← 型別不匹配！"A" 是 const char*（字串指標）， 無法直接賦值給 char，會導致編譯錯誤或警告！
