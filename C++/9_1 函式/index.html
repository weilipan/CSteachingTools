🔀 函式多載 (Function Overloading)

函式多載允許在同一個範圍內定義多個同名函式，只要它們的參數列表不同。編譯器會根據呼叫時提供的參數自動選擇正確的版本。

📋 多載規則

✅ 參數數量不同
✅ 參數型別不同
✅ 參數順序不同
❌ 僅回傳型別不同

💡 實際範例

範例 1 參數數量不同 
// 兩個參數
int add(int a, int b) {
    return a + b;
}

// 三個參數
int add(int a, int b, int c) {
    return a + b + c;
}

// 使用
int sum1 = add(5, 3);        // 呼叫第一個版本 → 8
int sum2 = add(5, 3, 2);     // 呼叫第二個版本 → 10
範例 2 參數型別不同 
// 整數版本
int max(int a, int b) {
    return (a > b) ? a : b;
}

// 浮點數版本
double max(double a, double b) {
    return (a > b) ? a : b;
}

// 字串版本
string max(string a, string b) {
    return (a > b) ? a : b;
}

// 使用
int maxInt = max(10, 20);           // → 20
double maxDouble = max(3.14, 2.71); // → 3.14
string maxStr = max("apple", "banana"); // → "banana"
範例 3 參數順序不同 
void display(int num, string text) {
    cout << "數字: " << num << ", 文字: " << text;
}

void display(string text, int num) {
    cout << "文字: " << text << ", 數字: " << num;
}

// 使用
display(42, "Hello");    // 呼叫第一個版本
display("World", 99);    // 呼叫第二個版本
錯誤範例 僅回傳型別不同（無效） 
// ❌ 編譯錯誤！不能僅靠回傳型別區分
int getValue() {
    return 42;
}

double getValue() {  // 錯誤：重複定義
    return 3.14;
}

// 編譯器無法判斷該呼叫哪一個
auto result = getValue();

✨ 多載的優點

  • 🎯 提高可讀性：使用相同名稱表達相同概念
  • 🔧 簡化介面：不需要記憶多個函式名稱
  • 💡 直覺使用：根據參數自動選擇正確版本
  • 🚀 程式碼重用：相似功能統一命名

核心概念

📤

傳值呼叫 (Pass by Value)

原始變數 x = 10
→ 複製 →
函式參數 a = 10 
void modify(int a) {
    a = 20; // 不影響原始變數
}
int x = 10;
modify(x); // x 仍為 10
🔗

傳址呼叫 (Pass by Reference)

原始變數 x = 10
→ 參照 →
函式參數 &a 指向 x 
void modify(int &a) {
    a = 20; // 會修改原始變數
}
int x = 10;
modify(x); // x 變為 20
🎯

遞迴 (Recursion)

factorial(4)
4 × factorial(3)
3 × factorial(2)
2 × factorial(1)
1 (基礎情況)
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

變數範圍 (Scope)

變數範圍決定了變數在程式中的可見性和生命週期。理解範圍對於避免命名衝突和記憶體管理至關重要。

🌍 全域範圍 (Global Scope)

定義位置：在所有函式外部宣告

可見範圍：整個程式檔案

生命週期：程式開始到結束

int globalCounter = 0; // 全域變數

void incrementCounter() {
    globalCounter++; // 可以存取
}

int main() {
    cout << globalCounter; // 可以存取
    incrementCounter();
    cout << globalCounter; // 輸出: 1
}
⚠️ 注意：過度使用全域變數可能導致程式難以維護和除錯

📦 區域範圍 (Local Scope)

定義位置：在函式內部宣告

可見範圍：僅在該函式內

生命週期：函式呼叫開始到結束

void functionA() {
    int x = 10; // functionA 的區域變數
    cout << x;  // 正確
}

void functionB() {
    int x = 20; // functionB 的區域變數（不同的 x）
    cout << x;  // 正確，輸出 20
    // 無法存取 functionA 的 x
}
💡 優點：不同函式可以使用相同的變數名稱而不衝突

🔲 區塊範圍 (Block Scope)

定義位置：在 { } 大括號內宣告

可見範圍：僅在該區塊內

生命週期：進入區塊到離開區塊

int main() {
    int x = 10;
    
    if (x > 5) {
        int y = 20;      // 區塊範圍變數
        cout << x << y;  // 正確：x=10, y=20
    }
    
    cout << x;  // 正確：x=10
    cout << y;  // ❌ 錯誤！y 不存在於此範圍
    
    for(int i = 0; i < 5; i++) {
        int z = i * 2;   // 每次迭代都重新建立
    }
    // i 和 z 在這裡都不可見
}

📥 參數範圍 (Parameter Scope)

定義位置：函式參數列表

可見範圍：整個函式內部

生命週期：函式呼叫期間

void calculate(int a, int b) {
    // a 和 b 是參數，在整個函式內可見
    int sum = a + b;
    
    if (sum > 10) {
        int result = sum * 2;
        // a, b, sum 都可以存取
    }
    // a, b, sum 可以存取，但 result 不行
}

🎭 變數遮蔽 (Variable Shadowing)

當內層範圍的變數名稱與外層相同時，內層變數會「遮蔽」外層變數

int x = 100;  // 全域變數

void testShadowing() {
    int x = 50;  // 區域變數，遮蔽全域 x
    cout << x;   // 輸出: 50
    
    {
        int x = 25;  // 區塊變數，遮蔽區域 x
        cout << x;   // 輸出: 25
    }
    
    cout << x;       // 輸出: 50（區域 x）
    cout << ::x;     // 輸出: 100（使用 :: 存取全域 x）
}

📊 範圍層級視覺化

全域範圍 int globalVar = 100;
函式範圍 (main) int localVar = 50;
if 區塊 int blockVar = 25;
✓ blockVar
✓ localVar
✓ globalVar
✗ blockVar
✓ localVar
✓ globalVar

✨ 最佳實踐建議

  • ✅ 優先使用最小範圍：變數應該在最小必要範圍內宣告
  • ✅ 避免全域變數：除非確實需要，否則使用區域變數
  • ✅ 使用有意義的名稱：避免因範圍不同而重複使用相同名稱
  • ✅ 在使用前宣告：盡量在變數使用位置附近宣告
  • ⚠️ 小心遮蔽：避免無意間的變數遮蔽造成混淆

🧮 數學運算

// 計算圓面積
double calculateArea(double radius) {
    const double PI = 3.14159;
    return PI * radius * radius;
}

// 使用範例
double area = calculateArea(5.0);
cout << "面積: " << area;

🔢 陣列處理

// 找出陣列最大值
int findMax(int arr[], int size) {
    int max = arr[0];
    for(int i = 1; i < size; i++) {
        if(arr[i] > max) max = arr[i];
    }
    return max;
}

// 使用範例
int numbers[] = {3, 7, 2, 9, 1};
int maxNum = findMax(numbers, 5);

📝 字串處理

// 反轉字串
string reverseString(string str) {
    string reversed = "";
    for(int i = str.length()-1; i >= 0; i--) {
        reversed += str[i];
    }
    return reversed;
}

// 使用範例
string original = "Hello";
string reversed = reverseString(original);

✅ 資料驗證

// 檢查是否為質數
bool isPrime(int n) {
    if(n <= 1) return false;
    for(int i = 2; i * i <= n; i++) {
        if(n % i == 0) return false;
    }
    return true;
}

// 使用範例
if(isPrime(17)) {
    cout << "17 是質數";
}
