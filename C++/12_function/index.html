⑥ 呼叫堆疊 (The Call Stack)

函式呼叫遵循 LIFO（後進先出）原則。每次呼叫，一個「執行紀錄」被推進堆疊頂端；函式結束後才彈出，執行點回到上一個函式。

int main() {
  funcA(); // ← 呼叫 A
  // A 執行完，繼續
  return 0;
}
 
void funcA() {
  funcB(); // ← 呼叫 B
  // B 執行完，繼續
}
 
void funcB() {
  // 最深層
}

📚 Call Stack

main()
funcA()
funcB()
↑ 頂端 (最新呼叫)
