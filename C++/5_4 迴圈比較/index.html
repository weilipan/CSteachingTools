C++ 迴圈大對決

視覺化理解 For, While, Do-While 的差異

(Try 5 to see the difference!)

For Loop (計次迴圈)

適用於 已知執行次數 的情況。

for (int i = 0; i < 5; i++) {
cout << "Count: " << i << endl;
}
i -
i < 5 ? -
Console Output

While Loop (前測迴圈)

適用於 未知次數，條件為真才執行。

int i = 0;
while (i < 5) {
cout << "Count: " << i << endl;
i++;
}
i -
i < 5 ? -
Console Output

Do-While Loop (後測迴圈)

保證 至少執行一次，之後再檢查條件。

int i = 0;
do {
cout << "Count: " << i << endl;
i++;
} while (i < 5);

i < 5 ? -
