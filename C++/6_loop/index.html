❤️ 2. while 迴圈（條件迴圈）—— HP 扣血機制

設定初始 HP，點擊「執行」。若 HP = 0，迴圈一次都不執行！觀察條件判斷先於一切。

初始 HP： 3
int hp = 3; while (hp > 0) {     hp--; // 扣血 } cout << "Game Over!" << endl;
— 設定 HP 並按執行 —

💥 3. 無窮迴圈大魔王（Infinite Loop）

漏寫 i++ 的慘烈後果！按下「執行」體驗失控感，再用「強制停止」解除危機。（本頁使用 setInterval 安全模擬，不會真的當機）

int i = 0; while (i < 5) {     cout << i << endl;     // i++; ← 忘記寫了！ }
⚠️ 無窮迴圈偵測！CPU 使用率飆升中…程式永不停止！

🎁 4. do-while 迴圈（保證執行一次）—— 盲盒抽卡機

先做再說！無論條件如何，至少抽一次。觀察執行順序：先抽卡 → 再判斷條件，與 while 形成強烈對比。

🚦 5. break vs continue（迴圈控制）

迴圈跑 1~5，第 3 回合遇到障礙。continue = 跳過這回合繼續；break = 砸碎整個迴圈逃跑！

for (int i = 1; i <= 5; i++) {     if (i == 3) {         /* break 或 continue */     }     cout << i << endl; }

↓
