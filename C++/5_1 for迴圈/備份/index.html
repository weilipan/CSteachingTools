C++ for Loop 視覺化詳解

掌握程式迴圈的核心邏輯，從初始化到條件終止。

1. 語法結構解析

for (初始化; 條件; 更新) {
    // 執行內容
}

初始化 (Initialization)

迴圈開始前僅執行一次，通常用於宣告並設定計數變數。

條件 (Condition)

每次進入迴圈前檢查。若為 true 則執行，false 則結束迴圈。

更新 (Update)

每次執行完迴圈本體後執行，通常用於增減計數器。

💡 專家提醒： 在 for 標頭中宣告的變數（如 int i）具有區塊作用域 (Block Scope)。這意味著一旦離開迴圈，該變數就會被銷毀，無法再被存取。
