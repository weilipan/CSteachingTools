C++ for Loop 視覺化詳解

掌握程式迴圈的核心邏輯：初始化 → 條件檢查 → 執行 → 更新

1. 語法結構解析

for (初始化; 條件; 更新) {
    // 執行內容 (Body)
}

初始化 (Initialization)

迴圈開始前僅執行 一次。通常用於宣告計數變數，如 int i = 0。

條件 (Condition)

每次進入迴圈前檢查。若為 true 才執行；若為 false 則跳出迴圈。

更新 (Update)

每次執行完「執行內容」後執行。通常用於增減計數器，如 i++。

2. 互動式模擬器

for (int i = 0; i < 5; i++) {
    cout << "Count: " << i << endl;
}
cout << "Done!";

3. 易犯錯誤 (Common Mistakes)

                

                    

                        
🚫 標頭後多加分號

                        
for(...); 會使迴圈本體變為空，大括號內容只會執行一次且 i 無法使用。

                    

                    

                        
♾️ 無限迴圈

                        
條件永遠為 true 或忘記更新變數（如 i-- 導致 i < 5 永遠成立）。

                    

                    

                        
⚠️ 差一錯誤 (Off-by-one)

                        
混淆 i < limit 與 i <= limit，導致多執行或少執行一次。

                    

                    

                        
📍 作用域問題

                        
在 for 標頭宣告的 i，離開迴圈後就消失了，無法再存取。
