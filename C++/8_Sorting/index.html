🔢 C++ 一維陣列：排序演算法
Sorting Algorithms — 互動視覺化學習

🫧 Module 1：氣泡排序 (Bubble Sort)

每輪掃描時比較相鄰兩個元素，若左 > 右則交換，最大值像泡泡一樣浮到最右端。時間複雜度 O(N²)。

點擊「開始排序」觀看動畫

🔍 Module 2：選擇排序 (Selection Sort)

每輪在未排序區間找最小值，再與未排序區最左端交換。共需 N-1 輪。時間複雜度 O(N²)。

點擊「開始排序」觀看動畫

📌 Module 3：插入排序 (Insertion Sort)

把第 i 個元素「抽出」，從左側已排序序列由右往左找到合適位置插入。就像整理撲克牌。時間複雜度 O(N²)。

點擊「開始排序」觀看動畫

⚡ Module 4：快速排序 (Quick Sort)

選定基準 (Pivot)，左右分區後遞迴。平均 O(N log N)，最壞 O(N²)。實務上速度最快之一。

點擊「開始排序」觀看動畫

🔀 Module 5：合併排序 (Merge Sort)

遞迴地切半直到剩 1 個，再合併。需要暫存空間。時間複雜度穩定 O(N log N)。

Temp Buffer（暫存區）
點擊「開始排序」觀看動畫

🪣 Module 6：基數排序 (Radix Sort)

非比較型排序，按位數（個位 → 十位 → 百位）放入對應桶子再取出。時間複雜度 O(d·N)。
