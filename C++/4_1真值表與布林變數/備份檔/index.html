什麼是布林變數？

布林（Boolean）變數是一種只能儲存兩種值的資料型態：true（真）或 false（假）。在 C++ 中，我們使用 bool 關鍵字來宣告布林變數。

// 宣告布林變數 bool isStudent = true; bool hasLicense = false; // 布林變數在記憶體中佔用 1 byte bool flag; // 未初始化的布林變數

常見用途

條件判斷

if (isStudent) { cout << "學生票價"; }

迴圈控制

while (gameRunning) { // 遊戲主迴圈 }

布林值的記憶體表示

在 C++ 中，布林值在記憶體中以整數形式儲存：

  • true 通常表示為 1
  • false 通常表示為 0
  • 任何非零值在布林上下文中都被視為 true
bool a = 5; // a 為 true (非零值) bool b = 0; // b 為 false bool c = -3; // c 為 true (非零值)

真值表 (Truth Tables)

真值表是一種用來表示邏輯運算結果的表格。它列出所有可能的輸入組合及其對應的輸出結果。

邏輯運算子 (Logical Operators)

C++ 提供三種主要的邏輯運算子，用於組合或修改布林運算式。

&& (邏輯 AND)

當且僅當兩個運算元都為 true 時，結果才為 true。

bool hasTicket = true; bool hasID = true; if (hasTicket && hasID) { cout << "可以入場"; }

✓ 兩個條件都必須滿足

|| (邏輯 OR)

只要有一個運算元為 true，結果就為 true。

bool isWeekend = false; bool isHoliday = true; if (isWeekend || isHoliday) { cout << "不用上班"; }

✓ 至少一個條件滿足即可

! (邏輯 NOT)

反轉布林值：true 變 false，false 變 true。

bool isRaining = false; if (!isRaining) { cout << "可以出門"; }

✓ 否定原本的條件

短路求值 (Short-Circuit)

邏輯運算子具有短路特性，可提升效能。

// && 短路：左邊為 false 就不檢查右邊 if (false && expensiveFunction()) { // expensiveFunction() 不會被呼叫 } // || 短路：左邊為 true 就不檢查右邊 if (true || expensiveFunction()) { // expensiveFunction() 不會被呼叫 }

關係運算子 (Relational Operators)

關係運算子用於比較兩個值，並返回布林結果。這些運算子在條件判斷中非常常用。

== (等於)

int x = 5; int y = 5; bool result = (x == y); // true

!= (不等於)

int x = 5; int y = 3; bool result = (x != y); // true

< (小於)

int x = 3; int y = 5; bool result = (x < y); // true

> (大於)

int x = 7; int y = 5; bool result = (x > y); // true

<= (小於等於)

int x = 5; int y = 5; bool result = (x <= y); // true

>= (大於等於)

int x = 7; int y = 5; bool result = (x >= y); // true

複合布林運算式

在實際程式設計中，我們經常需要組合多個條件來做出決策。這就需要使用複合布林運算式。

運算子優先順序

// 優先順序（由高到低）： // 1. ! (NOT) // 2. 關係運算子 (<, >, <=, >=, ==, !=) // 3. && (AND) // 4. || (OR) bool result = !false || true && false; // 等同於: (!false) || (true && false) // 等同於: true || false // 結果: true

實際應用範例

範例 1: 入場資格檢查

int age = 20; bool hasTicket = true; bool isVIP = false; // 18歲以上且有票，或是VIP bool canEnter = (age >= 18 && hasTicket) || isVIP; cout << canEnter; // true

範例 2: 成績評定

int score = 85; bool attendance = true; // 及格條件：分數>=60 且 出席率良好 bool passed = score >= 60 && attendance; // 優等：分數>=90 或 (分數>=80 且出席) bool excellent = score >= 90 || (score >= 80 && attendance);

範例 3: 範圍檢查

int temperature = 25; // 舒適溫度範圍：20-28度 bool comfortable = temperature >= 20 && temperature <= 28; // 極端溫度：<0 或 >40 bool extreme = temperature < 0 || temperature > 40;

範例 4: 使用括號明確化

bool a = true, b = false, c = true; // 不建議：容易混淆 bool result1 = a || b && c; // 建議：使用括號明確表達意圖 bool result2 = a || (b && c); bool result3 = (a || b) && c;

互動式程式範例

以下是一些實用的 C++ 程式範例，展示布林變數和邏輯運算的應用。

範例 1: 閏年判斷

#include <iostream> using namespace std; int main() { int year = 2024; // 閏年規則： // 能被4整除 且 (不能被100整除 或 能被400整除) bool isLeapYear = (year % 4 == 0) && (year % 100 != 0 || year % 400 == 0); if (isLeapYear) { cout << year << " 是閏年" << endl; } else { cout << year << " 不是閏年" << endl; } return 0; }

範例 2: 成績等第判定

#include <iostream> using namespace std; int main() { int score = 85; bool isA = score >= 90; bool isB = score >= 80 && score < 90; bool isC = score >= 70 && score < 80; bool isD = score >= 60 && score < 70; bool isF = score < 60; if (isA) cout << "等第: A" << endl; else if (isB) cout << "等第: B" << endl; else if (isC) cout << "等第: C" << endl; else if (isD) cout << "等第: D" << endl; else cout << "等第: F" << endl; return 0; }

範例 3: 三角形判斷

#include <iostream> using namespace std; int main() { int a = 3, b = 4, c = 5; // 三角形成立條件：任兩邊和大於第三邊 bool isTriangle = (a + b > c) && (b + c > a) && (a + c > b); if (isTriangle) { cout << "可以形成三角形" << endl; } else { cout << "無法形成三角形" << endl; } return 0; }

範例 4: 密碼強度檢查

#include <iostream> #include <string> using namespace std; int main() { string password = "Abc123!@"; bool hasLength = password.length() >= 8; bool hasUpper = false; // 簡化示範 bool hasLower = false; bool hasDigit = false; bool hasSpecial = false; // 強密碼：長度>=8 且 包含多種字元 bool isStrong = hasLength && hasUpper && hasLower && hasDigit; return 0; }
