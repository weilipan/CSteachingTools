深入理解條件判斷的運作原理

📌 if 語句

概念說明

if 語句是最基本的條件判斷結構，當條件為真（true）時，執行特定的程式碼區塊。

語法結構

if (條件) {
    // 條件為真時執行的程式碼
}

流程圖

範例：判斷數字是否為正數

#include <iostream>
using namespace std;

int main() {
    int number = 10;
    
    if (number > 0) {
        cout << number << " 是正數" << endl;
    }
    
    return 0;
}

💡 重點提示

  • 條件必須是布林值（true 或 false）
  • 如果只有一行程式碼，大括號可以省略（但不建議）
  • 條件可以使用比較運算子：>、<、>=、<=、==、!=

📌 if-else 語句

概念說明

if-else 語句提供兩個執行路徑：當條件為真時執行 if 區塊，為假時執行 else 區塊。

語法結構

if (條件) {
    // 條件為真時執行
} else {
    // 條件為假時執行
}

流程圖

範例：判斷奇數或偶數

#include <iostream>
using namespace std;

int main() {
    int number = 7;
    
    if (number % 2 == 0) {
        cout << number << " 是偶數" << endl;
    } else {
        cout << number << " 是奇數" << endl;
    }
    
    return 0;
}

💡 重點提示

  • 必定會執行其中一個區塊（if 或 else）
  • 適合用於二選一的情況
  • else 區塊不需要條件

📌 else-if 語句

概念說明

else-if 語句允許檢查多個條件，依序判斷直到找到第一個為真的條件。

語法結構

if (條件1) {
    // 條件1為真時執行
} else if (條件2) {
    // 條件2為真時執行
} else if (條件3) {
    // 條件3為真時執行
} else {
    // 所有條件都為假時執行
}

流程圖

範例：成績等級判斷

#include <iostream>
using namespace std;

int main() {
    int score = 85;
    
    if (score >= 90) {
        cout << "等級：A (優秀)" << endl;
    } else if (score >= 80) {
        cout << "等級：B (良好)" << endl;
    } else if (score >= 70) {
        cout << "等級：C (普通)" << endl;
    } else if (score >= 60) {
        cout << "等級：D (及格)" << endl;
    } else {
        cout << "等級：F (不及格)" << endl;
    }
    
    return 0;
}

💡 重點提示

  • 條件由上而下依序檢查
  • 只要有一個條件為真，就不會檢查後續條件
  • 最後的 else 是可選的
  • 適合用於多個互斥條件的判斷

📌 switch 語句

概念說明

switch 語句根據變數的值，跳轉到對應的 case 標籤執行。適合用於多個固定值的比較。

語法結構

switch (變數) {
    case 值1:
        // 變數等於值1時執行
        break;
    case 值2:
        // 變數等於值2時執行
        break;
    default:
        // 沒有匹配的case時執行
        break;
}

流程圖

範例：星期判斷

#include <iostream>
using namespace std;

int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            cout << "星期一" << endl;
            break;
        case 2:
            cout << "星期二" << endl;
            break;
        case 3:
            cout << "星期三" << endl;
            break;
        case 4:
            cout << "星期四" << endl;
            break;
        case 5:
            cout << "星期五" << endl;
            break;
        case 6:
            cout << "星期六" << endl;
            break;
        case 7:
            cout << "星期日" << endl;
            break;
        default:
            cout << "無效的日期" << endl;
            break;
    }
    
    return 0;
}

🔄 switch vs if-else

switch 語句

  • ✅ 適合固定值的比較
  • ✅ 程式碼更簡潔易讀
  • ✅ 執行效率較高
  • ❌ 只能比較整數、字元
  • ❌ 不能使用範圍判斷

if-else 語句

  • ✅ 可以使用任何條件
  • ✅ 支援範圍判斷
  • ✅ 更靈活
  • ❌ 多條件時較冗長
  • ❌ 執行效率稍低

💡 重點提示

  • break 用於跳出 switch，否則會繼續執行下一個 case（fall-through）
  • default 是可選的，用於處理所有未匹配的情況
  • case 的值必須是常數（不能是變數）
  • switch 只能用於整數型別（int、char）和列舉型別

📌 巢狀判斷結構

概念說明

巢狀判斷是指在一個判斷結構內部包含另一個判斷結構，用於處理更複雜的條件邏輯。

語法結構

if (外層條件) {
    if (內層條件1) {
        // 外層和內層條件1都為真
    } else {
        // 外層為真，內層條件1為假
    }
} else {
    if (內層條件2) {
        // 外層為假，內層條件2為真
    }
}

流程圖

範例：閏年判斷

#include <iostream>
using namespace std;

int main() {
    int year = 2024;
    bool isLeapYear = false;
    
    if (year % 4 == 0) {
        if (year % 100 == 0) {
            if (year % 400 == 0) {
                isLeapYear = true;  // 能被400整除
            } else {
                isLeapYear = false; // 能被100整除但不能被400整除
            }
        } else {
            isLeapYear = true;      // 能被4整除但不能被100整除
        }
    } else {
        isLeapYear = false;         // 不能被4整除
    }
    
    if (isLeapYear) {
        cout << year << " 是閏年" << endl;
    } else {
        cout << year << " 不是閏年" << endl;
    }
    
    return 0;
}

💡 重點提示

  • 巢狀層數不宜過多（建議不超過3層），否則程式碼難以閱讀
  • 適當使用邏輯運算子（&&、||）可以減少巢狀層數
  • 注意縮排，保持程式碼清晰
  • 複雜的巢狀判斷可以考慮拆分成多個函式

🔄 簡化版本（使用邏輯運算子）

#include <iostream>
using namespace std;

int main() {
    int year = 2024;
    bool isLeapYear = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
    
    if (isLeapYear) {
        cout << year << " 是閏年" << endl;
    } else {
        cout << year << " 不是閏年" << endl;
    }
    
    return 0;
}

📌 實際應用範例

應用一：簡易計算機

#include <iostream>
using namespace std;

int main() {
    double num1, num2;
    char op;
    
    cout << "輸入第一個數字: ";
    cin >> num1;
    cout << "輸入運算子 (+, -, *, /): ";
    cin >> op;
    cout << "輸入第二個數字: ";
    cin >> num2;
    
    switch (op) {
        case '+':
            cout << "結果: " << num1 + num2 << endl;
            break;
        case '-':
            cout << "結果: " << num1 - num2 << endl;
            break;
        case '*':
            cout << "結果: " << num1 * num2 << endl;
            break;
        case '/':
            if (num2 != 0) {
                cout << "結果: " << num1 / num2 << endl;
            } else {
                cout << "錯誤: 除數不能為0" << endl;
            }
            break;
        default:
            cout << "錯誤: 無效的運算子" << endl;
            break;
    }
    
    return 0;
}

應用二：選單系統

#include <iostream>
using namespace std;

int main() {

            break;
        case 4:
            cout << "執行：刪除資料功能" << endl;

            cout << "感謝使用，再見！" << endl;
            break;

            cout << "錯誤：無效的選項" << endl;

    return 0;
}

應用三：資格判斷系統

#include <iostream>
using namespace std;

int main() {

    bool hasLicense;
    int drivingYears;
    
    cout << "輸入年齡: ";

        cout << "不符合資格：沒有駕照" << endl;

        cout << "輸入駕駛年資: ";
        cin >> drivingYears;
        
        if (drivingYears < 1) {
            cout << "資格：新手駕駛" << endl;
        } else if (drivingYears < 3) {
            cout << "資格：一般駕駛" << endl;
        } else if (drivingYears < 10) {
            cout << "資格：經驗駕駛" << endl;
        } else {
            cout << "資格：資深駕駛" << endl;
        }
    }
    
    return 0;
}

📚 總結

判斷結構是程式設計的基礎，選擇適當的判斷結構可以讓程式碼更清晰、更有效率：

  • if：單一條件判斷
  • if-else：二選一的情況
  • else-if：多個互斥條件
  • switch：多個固定值的比較
  • 巢狀判斷：複雜的條件組合
