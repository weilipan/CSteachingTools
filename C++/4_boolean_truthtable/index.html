⚖️ 2. 關係運算子 (Relational Operators)

==, !=, >, <, >=, <=  |  即時比較與常見錯誤說明

⚠️ 初學者常犯錯誤：搞混 == 與 =
==
if (a == b)
比較運算子：詢問「a 是否等於 b？」
運算結果是 bool 型態（true 或 false）。
=
a = b;
賦值運算子：把 b 的值「存進」a 裡面。
這不是比較！常見 Bug 來源：誤打成 if (a = b)
